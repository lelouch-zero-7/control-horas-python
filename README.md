# Sistema de Control de Horas

Programa en Python que permite calcular las horas trabajadas por un equipo durante la semana y clasificar su jornada laboral.

## Funcionalidades

- Cálculo del total de horas trabajadas
- Clasificación de jornada (Sobretiempo o Estándar)
- Menú interactivo con opciones
- Validación de entrada del usuario
